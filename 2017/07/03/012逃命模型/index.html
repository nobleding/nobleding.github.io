逃命模型

逃命模型简单来说：碰到老虎追击的时候，人们本能地选择逃命。推而广之：人们总是希望尽快摆脱危险/不舒服的环境。

人很多时候的自我欺骗都因此而起，细化来看——我们的拖延症：把责任/危险/不舒服推卸给未来那个陌生的自己，或者一个团队中推卸责任、标榜功劳，或是每次失败/妥协/醉酒/游戏之后的自责以及着手制定宏伟计划/人生目标，都是一样的套路，它都在创造这样一个事实——让此刻的自己尽快摆脱不舒服，从而感觉良好。 比如去学习一项技能，总有各种理由来拖延，这会让自己在这个过程中更舒服;而一旦最终没有完成预定的目标，自己又立刻采取新的措施——赶快自责，然后制造一种“发奋图强”的假象——这让此刻的自己感觉高大了不少，至少心怀希望，而不是一个真正的失败者，虽然自始至终自己都没有采取实际的行动。

1、逃命模型与三个自己

逃命模型里有三个自己：历史的思维模型、当下的自己以及未来的自己，很多冲突也因此而起。很多事物都有三个自己的冲突，不仅仅限于逃命。拿单位的发展来讲，问题有历史原因形成，也有当下的自己/环境造就，这帮领导是希望单位长久发展还是任期内不出差错就好？对未来的定位不同方法也就不同，再比如疾病，历史思维模型：慢性疾病，当下的自己：突发疾病，未来的自己：长久医治还是短暂治疗？ 定位不同方法也就不同（很多老人都拒绝治疗，以期体面一些地离去）。

2、逃命模型与短期记忆

逃命模型让人们这样一些行为：当有压力、恐惧、失落等情绪时，人们本能地想逃离这个情境，而逃离的一个举措就是去做那些自己能全身心投入的事情或让自己神经麻痹的事情，比如酗酒、抽烟，或是去一些热闹场所吃吃喝喝、纸醉金迷。

有些问题是无法通过自己的努力去改善，这也是为什么很多铁骨铮铮的汉子为一些亲人的离去而落泪。但大部分问题并非如此，人更容易被眼前的事物所吸引：当脑子里有一些杂思乱想时，一般人可能会这么做：怎么可以有这样的念头？！你是不是有病呀？！父母那么辛苦你却整天在想这些？诸如此类，很多时候念头来了有三个阶段：1）念头来了；2）意识到念头来了；3）批评自己不该有这个念头。其实脑子里意识到这个念头，这个念头已经在淡化了，何必再多此一举？如果让某人想一只大象，最好的办法是让他不想大象。很多烦恼一旦意识到，就已经淡化了，如果再批评自己不该有此烦恼，就有些庸人自扰——死死抓着这个念头不放岂不糊涂？这个时候如果意识到逃命模型与短期记忆，可以这样做：例如脑子里总想着某一个人，可以去看一些风景，如果自己有沉迷的兴趣——下棋、绘画或者舞蹈，那就去做（只要是能吸引自己，让自己全身心投入的事情都可以）。可能几个小时都用不着，这个念头早已经忘记得一干二净了。

很多时候人们通过酗酒、刷剧、热闹场所的欢聚等等，都不过是短期记忆与逃命模型的应用，只不过一些措施会让自己的境况变得更差。古人大悲大喜时借物抒情，成就了古诗词；今人多有吸毒、酗酒，坏掉了身体。目的很多都是一致，措施上有了差别。一个美好的兴趣对于人的一生是多么重要。

一个身边的例子：如果你让每个人民掏出几块钱，这都会让他们情绪很反动，因为这是动的他们眼前的钱。但如果你从他们下个月的工资里抽取税收，即使这样的付出要多一些，他们也可能会心安理得地接受，甚至有点像这些拿去的钱本就与他们无关。请别人帮忙或者自己的一些行为需要他人割舍一些利益，最好先提前说一下（例如需要别人来开一个会，最好提前通知并说定日期），因为他们未来的时间对于当下的自己，相比来讲可能不会看得那么重要。

该点的反面

人都有一种本能:尽快逃离让自己不舒服的环境。反过来也成立：尽可能不离开让自己舒服的环境。

例如跟某人聊天聊的还不错（也许只是我自己这么觉得），是因为两人之间没有指责或者嘲笑，更多的是赞美、认同或者理解，创造的是一种相对和气的氛围。所以反过来我要是挖苦对方、嘲笑对方、鄙视对方，或者对方反过来如此待我，不出三天，友尽。

这也是沟通的一般技巧，很多人深谙此道。以前看到一个故事：一群孩子老是来一位老人的住所旁，扰了他的清净，于是他想出来一个办法：每次来都给孩子糖吃，之后突然有一天，不再提供糖果，孩子们也不再过来。

再比如：如果留心观察，会发现我们更乐意同那些给予我们正反馈的人、事、物呆在一起。同样，如果希望同某人亲近，努力创造这样的氛围。不过需要记住一点：讨价挑刺的是买家，说好的往往都是过客。

如果你想远离某人，那么努力创造负反馈的环境。这样还不够，最好能再加一个对比：（正反馈最好不要如此，破镜难圆，友尽难续）先对ta创造一个正反馈的环境，再创造一个负反馈的环境。古人的话就是：将欲取之，必先予之。前面那位老人的例子就很好地说明了这一点。

3、逃命模型与以人为本

有一句话概括了这个观点：人之恶在好为人师。自我批评也好、建议他人也罢，诉诸利益好过解释道理。很多时候都在跟自己讲道理：应该早起、应该健身、应该学习新的技能、应该多接触接触人群......却很少跟自己诉诸利益：如果健身一周，奖励自己一个陶笛。诉诸利益也未尝不是一份关怀，它包含着理解之同情以及换位思考的理性。

说三个小例子。

例1：推荐书籍

放在沟通上来讲，如果想给他人推荐一本书，可以：

  • 采取分享的口吻：告诉别人自己如何如何受启发，这本书如何如何不错
    也可以
  • 采取请教的口吻：关于这本书有哪些章节不明白，希望对方帮忙解答或是提供一些思考的角度

采取请教的口吻就是站在对方的角度诉诸利益，虽然自己的初衷是与对方分享一些自认为对ta有益的知识。

例2：辅导功课

放在家庭教育上来讲，家长虽然懂得多一些，但家长辅导孩子做题的做法，可能并不比家长以请教的口吻，让孩子辅导自己知识来得高明。后者也是采取了请教的口吻，采取请教的口吻就是站在对方的角度诉诸利益。

例3：寻求帮助

1） 生理需求的物质交换，诉诸利益的体现可以是让对方得到实惠。
2）社会、尊重需求的物质交换，诉诸利益的体现可以是让对方得到认可、赞美、价值体现。上面例子的请教的口吻都属于这一层。

这两点是反过来的，因为物质往往有限，而精神则趋于无限（不用破财也不用出力）。因为物质有限，所以更多的人希望可以被赠予；因为精神无限，所以更多的人希望去赠予。凡是不希望的，大部分人都想避开它，这也是“逃命模型”的体现。很多人在沟通、合作等方面，都没有注意这一点，包括我自己。

从小我受到的家庭教育就是不要拿别人的一针一线，包括到今天，我仍然非常不习惯去麻烦别人。这一点是物质匮乏年代的修养，但今天物质已经比较富足，观念是不是要有所改变？管子说过衣食足而知荣辱，仓廪实而知礼节。马斯洛的需求分析生理需求→安全需求→社会需求→尊重需求→自我实现。不拿一针一线停留在生理需求的社会里，但如今的人更关注荣辱、礼节，这就好比好为人师的教导，比不过以请教为口吻的倾听。

经常听一些人这么表述：xxx白眼狼，我帮他他还对不起我。 xxx可是我一手提拔的，如今却过河拆桥。 我帮了xxx不少忙，他怎么一点也不知道感恩呢？明不明白问题出在了哪里？犯错的可怕之处在于我们以为我们在坚持正确的事情。说的极端一点：如果你想亲近一个人，那么让ta帮你一个忙; 如果你想远离一个人，那么去帮ta一个忙。这些忙更多是指生理需求以外的交换了。
